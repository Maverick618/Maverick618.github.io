# Latex 一些语法

### 特殊符号
- 花体大写字母，如 $\mathcal{X, L}$。语法：`\mathcal{}`
- 双线字母，如 $\mathbb{P, N}$。语法：`\mathbb{}`
- 用于区分1的花体(不知道是不是) $\ell$。语法：`\ell`
- 特殊章节符号 $\S$。语法：`\S`
- ==双括号==，$(\!\lvert P \lvert\!)$ 或 $[\![  ]\!]$。语法`(\!\lvert P \lvert\!)`或 `[\![  ]\!]`。
*感觉不如导入别的package。*
